# ANN: Approximate Nearest Neighbors

Copy from https://www.cs.umd.edu/~mount/ANN/


## Usage

### Using `make`

```bash
git clone https://github.com/long-gong/ANN.git
cd ANN
make
```

### Using 'cmake'

```bash
git clone https://github.com/long-gong/ANN.git
cd ANN
mkdir build && cd build
cmake ..
make
```
